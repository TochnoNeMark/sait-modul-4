# OpenWing — сайт авиакомпании

Учебный многостраничный адаптивный сайт вымышленной авиакомпании **OpenWing**.
Чистые HTML, CSS и JavaScript — без сборщиков, фреймворков и внешних зависимостей.
Хостинг статический, совместим с GitHub Pages.

## Возможности

- **Главная** — промо-баннер тарифов и сетка направлений.
- **Билеты** — форма с пятью режимами: покупка билета или отеля (выпадающие списки аэропортов, счётчик гостей, календарь), регистрация, бронирование места, статус рейса.
- **События** — анонс курсов пилотов с переходом к записи.
- **Запись на курсы** — отдельная форма заявки.
- **Отзывы** — отзыв со звёздным рейтингом и бесконечная лента.
- **Мерч** — каталог с категориями (чемоданы, подушки, электроника), карточка товара с выбором цвета.
- **Корзина** — позиции, счётчик количества и сумма (хранится в `localStorage`).
- **О нас** — блок о компании.
- **Тарифы / OpenWing+** — демо-подписка и каталог услуг (места, питание, дополнительно) с отдельной корзиной, правилами добавления и модалкой «Подробнее».
- **404** — кастомная страница ошибки: композиция «4 ⬭ 4» с анимацией облаков в иллюминаторе и случайной подписью.

Все формы интерактивны, но демонстрационные — ничего не отправляется на сервер и не оплачивается.

## Технологии

- HTML5, CSS3 (flex/grid, `clamp()` для адаптива, CSS-переменные), ванильный JavaScript (ES2015+).
- Шрифт — **Montserrat Alternates** (Google Fonts).
- Адаптив рассчитан на три ширины: десктоп (~1440), планшет (~820), мобайл (~390).
- Без сборки: любой `*.html` открывается в браузере напрямую.

## Структура

```
├── *.html                страницы разделов (index, tickets, events, reviews,
│                          merch, product, enroll, cart, about, pricing, 404)
├── styles/               reset.css · fonts.css · main.css · adaptive.css
│                          (точка входа — style.css, импортирует остальные по порядку)
├── scripts/              JS-модули по страницам (catalog, main, booking, reviews,
│                          merch, product, cart, pricing, enroll, notfound)
└── assets/               изображения (WebP) и PDF
```

CSS построен на дизайн-токенах (цвета и единые размеры шрифта в `:root`), desktop-first; медиазапросы вынесены в `adaptive.css`.

## Запуск локально

Достаточно открыть `index.html` в браузере. Либо поднять простой статический сервер:

```bash
python -m http.server 8000
# затем открыть http://localhost:8000
```

(На Windows вместо `python` может использоваться `py`.)
